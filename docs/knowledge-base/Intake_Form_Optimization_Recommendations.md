Recommended Form Structure
Option A: Two-Step Type Selection
Step 1: What do you need?

	•	Review/Edit existing content
	•	Create new content
	•	Strategy or consultation
	•	Event support
	•	Governance/Mandate support
	•	General inquiry

Step 2 (Conditional): What type of content?

If "Review/Edit existing content":

	•	Email draft
	•	Announcement draft
	•	Newsletter contribution
	•	Website page
	•	Presentation/slides
	•	Other document

If "Create new content":

	•	Email/announcement (net new)
	•	Newsletter edition
	•	Website page/section
	•	Playbook or strategy doc
	•	Templates
	•	Video content

If "Strategy or consultation":

	•	Comms strategy
	•	Channel/audience advice
	•	Messaging guidance
	•	Process improvement

If "Event support":

	•	Speaker recruitment
	•	Event comms (invites/reminders)
	•	Event content (abstracts, descriptions)

If "Governance/Mandate support":

	•	New mandate intake
	•	Approval coordination
	•	TRR support


Option B: Single Enhanced Dropdown
Type of Request:

	•	Quick Review — Edit my existing draft (email, announcement, doc)
	•	Website Update — Update or add content to an existing page
	•	Newsletter Contribution — Submit content for upcoming newsletter
	•	New Content Creation — Need net-new content created (email, playbook, strategy)
	•	Event Comms — Invitations, speaker recruitment, event promotion
	•	Mandate/Governance — Approval process, TRR, policy comms
	•	Executive Comms — Content for/from L8+ leadership
	•	Strategy Consultation — Advice on approach, channels, or messaging
	•	General Inquiry — Other

Why this works: Categories now signal touch level:

	•	1-3 → Likely Low Touch
	•	4-6 → Likely Medium Touch
	•	7-8 → Likely High Touch
	•	9 → Triage required


Additional Fields to Add
Based on report gaps, these fields would improve triage:
1. Scope (NEW — High Value)
Who is the primary audience?

	•	My immediate team only
	•	Multiple teams within [TEAM_NAME]
	•	Cross-org (outside [TEAM_NAME] but within the organization)
	•	External (outside the organization)
	•	Core Leadership (Directors/VPs)
	•	L8+ Senior ICs (PE/DE/Fellows)

Rationale: Scope was a consistent driver of touch level in the report.


2. Content Status (NEW — High Value)
What's the current state?

	•	Draft ready for review
	•	Outline/rough notes only
	•	Nothing yet — need content created
	•	Existing content needs restructuring

Rationale: Distinguishes edit vs. create — the primary triage rule.


3. Urgency Clarification (MODIFY Existing)
Current field: "Is there a specific date this needs to go out?"

Recommended change: Make it structured:

	•	No hard deadline — flexible
	•	Within 2 weeks
	•	Within 1 week
	•	Within 2-3 days
	•	Urgent — within 24-48 hours
	•	Emergency — same day (explain in notes)

Rationale: Open text field often left blank or vague. Structured options force clarity.


4. Leadership Visibility (NEW — Medium Value)
Will this be sent by or seen by L8+ leadership?

	•	Yes — sent by/to VP or Director
	•	Yes — sent by/to PE, DE, or Fellow
	•	Yes — [VP_NAME] involved
	•	No — standard distribution

Rationale: Leadership visibility consistently triggered High Touch in the report.


5. Previous Support (NEW — Low Value but Useful)
Have you worked with [TEAM_NAME] Comms on this topic before?

	•	Yes — this is a follow-up or recurring request
	•	No — this is new

Rationale: Recurring requests (newsletters, mandates) have established patterns. New requests may need more discovery.
